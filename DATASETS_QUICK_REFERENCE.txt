NSL-KDD:

UNSW-NB15: ⭐ BEST MATCH!
   ✓ Normal

CIC-IDS-2017: ⭐ PERFECT MATCH!

   ✓ Brute Force (Tuesday FTP/SSH, Thursday Web)
   ✓ SQL Injection (Thursday web attacks)
   + DoS, XSS, Heartbleed, Infiltration

┌─────────────────────────────────────────────────────────────────────────────┐
│                           TROUBLESHOOTING                                   │
└─────────────────────────────────────────────────────────────────────────────┘

Problem: Download fails
Solution: Try manual download from links above

Problem: Files too large
Solution: Start with NSL-KDD (only 20 MB)

Problem: Training too slow
Solution: Use smaller batch_size or fewer samples

Problem: Out of memory
Solution: Reduce batch_size to 16 or use data subsampling

Problem: Dataset format doesn't match
Solution: Your data_loader.py should auto-detect format
